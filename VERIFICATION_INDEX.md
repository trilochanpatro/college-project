4. Import database schema

### Path 2: Oracle Cloud Always Free (Most Powerful)
1. Create Oracle Cloud account
2. Launch Linux VM instance
3. Install LAMP stack
4. Upload project files
5. Configure MySQL database
6. Set up Apache virtual host
7. Configure security groups

See: **HOSTING_GUIDE.md** → Oracle Cloud Section

---

### Path 3: Railway.app (Modern)
1. Connect GitHub account
2. Deploy from repository
3. Add MySQL database plugin
4. Configure environment variables
5. Deploy automatically

See: **HOSTING_GUIDE.md** → Railway Section

---

1. Install Raspberry Pi OS
2. Install LAMP stack
3. Upload project files
4. Configure database
